🥇 黃金核心公式（所有題共用）

直接回答 ＋ 補一句（理由 / 頻率 / 地點 / 對象）

A2 口說 58% 的題目會接「Vertel ook… / Zeg ook…」要你多講一句。所以養成反射：不管什麼題，一律答 2 句。第一句先答，第二句用 want（因為） 補理由最穩 —— want 後面語序不用倒裝，最不會錯。

1看圖描述 68 題 · 最常考

怎麼認出
開頭 Kijk naar het plaatje / de plaatjes，問 Wat doet hij/zij?
反射句架
Hij/Zij [動詞] [在哪 in/op/bij…]. Volgens mij [你的看法/理由].
立刻能說
Zij loopt op straat. Haar auto is kapot.
她在街上走。她的車壞了。
Hij gaat met de fiets naar zijn werk.
他騎腳踏車去上班。
🔁 反射順序：誰 → 做什麼(動詞) → 在哪(in/op/bij) → 補一句（為什麼 / volgens mij…）。看不懂圖也先講「Ik zie een man/vrouw. Hij/Zij …」

2二選一比較 17 題

怎麼認出
出現 liever（比較喜歡）或 A of B（…還是…）
反射句架
Ik [動詞] liever [A] dan [B], want [理由].
立刻能說
Ik ga liever naar een concert dan naar de bioscoop, want dat is gezelliger.
我比較喜歡去演唱會勝過電影院，因為比較熱鬧。
🔁 先選一邊 → liever A dan B → want 理由。不用兩邊都講。

3內容 Wat…? 45 題

怎麼認出
Wat doet u…? / Wat vindt u…? / Wat eet/maakt u…?
反射句架
Ik [動詞] [受詞]. / Ik vind [X] leuk.
立刻能說
Ik kook graag pasta. Ik vind dat lekker.
我喜歡煮義大利麵。我覺得很好吃。
🔁 Wat = 講「我做什麼 / 我覺得什麼」+ 一個感受詞（leuk / lekker / mooi）。

4頻率 Hoe vaak…? 14 題

怎麼認出
Hoe vaak …? （多常）
反射句架
Ik [動詞] [één/twee/drie] keer per [dag/week/maand/jaar].
立刻能說
Ik sport twee keer per week.
我一週運動兩次。
🔁 數字 + keer per + 時間單位。也可用 elke dag（每天）、soms（有時）、nooit（從不）。

5地點 Waar…? 16 題

怎麼認出
Waar …? （在哪 / 去哪）
反射句架
Ik [動詞] in / op / bij / naar [地點].
立刻能說
Ik eet het liefst thuis. / Ik werk in een ziekenhuis.
我最喜歡在家吃。/ 我在醫院工作。
🔁 介系詞口訣：在裡面 in、在表面/市場 op、在某機構 bij、去某處 naar。

6選擇 Welk(e)…? 13 題

怎麼認出
Welke / Welk …? （哪一個 / 哪些）
反射句架
Ik houd van [X]. / Ik kies [X], want [理由].
立刻能說
Ik houd van popmuziek. Bijvoorbeeld Coldplay.
我喜歡流行樂，例如 Coldplay。
🔁 挑一個具體的 + 可加 bijvoorbeeld（舉例）讓答案更滿。

7原因 Waarom…? 12 題

怎麼認出
Waarom …? 或追問 En waarom?
反射句架
[答案], want [主詞 + 動詞 + 其他].
立刻能說
Ik fiets graag, want dat is gezond.
我喜歡騎車，因為很健康。
🔁 want（語序不變）比 omdat（要倒裝）安全。萬用理由：want dat is gezond / leuk / handig / goedkoop / lekker。

8時間 Wanneer / Hoe laat…? 8 題

怎麼認出
Wanneer（何時）/ Hoe laat（幾點）
反射句架
Om [數字] uur. / In de ochtend · 's middags · 's avonds.
立刻能說
Ik begin om negen uur. / Ik werk het liefst in de ochtend.
我九點開始。/ 我最喜歡早上工作。
🔁 幾點用 om … uur；時段用 in de ochtend / middag、's avonds / 's nachts。

9類型 Wat voor…? 6 題

怎麼認出
Wat voor …? （什麼樣的 / 哪種）
反射句架
Ik [動詞/houd van] [種類]. Bijvoorbeeld [例子].
立刻能說
Ik ben schoonmaker. / Ik maak het liefst Aziatisch eten.
我是清潔工。/ 我最喜歡做亞洲菜。
🔁 Wat voor = 講「哪一種」+ 舉一個例子。

10方式 Hoe…? 6 題

怎麼認出
Hoe doet/reist/maakt u…? （怎麼做）
反射句架
Ik [動詞] met de [fiets/auto/trein]. / Eerst … dan …
立刻能說
Ik ga met de fiets. / Ik kook eerst de pasta, dan doe ik er groente bij.
我騎腳踏車去。/ 我先煮麵，再加蔬菜。
🔁 交通用 met de + 工具；步驟用 eerst … dan …。

11對象 Met wie…? 3 題

怎麼認出
Met wie …? （和誰）
反射句架
Ik [動詞] samen met mijn [家人/朋友].
立刻能說
Ik eet meestal samen met mijn familie.
我通常和家人一起吃。
🔁 samen met mijn + man / vrouw / kinderen / vrienden / familie。

12表態認同 Vindt u … ook? 2 題

怎麼認出
Vindt u … ook? / Bent u ook …?（你也…嗎）
反射句架
Ja, ik vind [X] ook leuk. / Nee, ik vind [X] niet leuk.
立刻能說
Ja, ik vind cadeautjes ook leuk. Ik krijg graag bloemen.
對，我也喜歡禮物。我喜歡收到花。
🔁 先 Ja/Nee 表態 → 重複關鍵字 + ook/niet → 補一個例子。

⚡ 反射加分小詞（塞進任何答案）

程度：het liefst（最愛）· graag（喜歡）· meestal（通常）· vaak（常）· soms（有時）· bijna altijd（幾乎總是）· nooit（從不）
連接：want（因為）· en（和）· maar（但是）· ook（也）· dan（然後）· bijvoorbeeld（例如）
表態：Ik vind … (niet) leuk / mooi / lekker / gezond / handig / gezellig / duur / goedkoop
萬用理由：…, want dat is gezond / leuk / handig / lekker / goedkoop / belangrijk / rustig
