🗣 荷蘭文口說考古題講義

Ad Appel「Spreekvaardigheid A2 Oefenexamen」全 16 份 · 138 題問答 + 答題模板

每題標 M? = 用到的模板。問句常先鋪陳情境，重點聽最後那個問句。

怎麼用：先背熟下面 9 個答題模板 → 展開任一份考卷，遮住答案，套模板自己講 2-3 句 → 再對照範例答案。所有考卷都是同樣套路。
📐 答題模板（先背這 9 個）
M0 核心原則：答 2–3 句

A2 口說不能只答一句。先直接回答 → 再補一句細節/理由。考官要聽到你能展開。

🇳🇱 Hoe gaat u op vakantie? → Ik ga liever met de trein. Ik reis nooit met het vliegtuig.

你怎麼去度假？→ 我比較喜歡搭火車。我從不搭飛機。

M1 Hoe vaak…?（多常）→ 頻率詞

用 elke dag / iedere week / vaak / soms / nooit / één keer per week。

🇳🇱 Luistert u vaak naar muziek? → Ik luister elke dag naar popmuziek.

你常聽音樂嗎？→ 我每天聽流行音樂。

M2 het liefst / liever（偏好）

問「最喜歡怎樣」用 het liefst（最…）或 liever … (dan)（比較喜歡）。

🇳🇱 Hoe gaat u het liefst op vakantie? → Ik ga het liefst met de auto.

你最喜歡怎麼去度假？→ 我最喜歡開車去。

M3 Wat wilt u / Wil…? → ik wil graag

講想要/願望用 Ik wil graag … 或 Ik zou graag … willen。

🇳🇱 Welke stad wilt u bezoeken? → Ik wil graag naar Parijs. Ik wil daar de Eiffeltoren zien.

你想去哪個城市？→ 我想去巴黎，想在那看艾菲爾鐵塔。

M4 Vindt u…?（覺得）→ ik vind … + want

表達意見用 Ik vind … + 形容詞，再用 want / omdat 給理由。

🇳🇱 Vindt u Nederlands moeilijk? → Ik vind Nederlands heel moeilijk, want de grammatica is lastig.

你覺得荷蘭語難嗎？→ 我覺得很難，因為文法很麻煩。

M5 Waarom…? → om…te / omdat

講目的用 om + 目的 + te + 原形；講原因用 omdat …。

🇳🇱 Waarom sport u? → Ik sport om gezond te blijven.

你為什麼運動？→ 為了保持健康。

M6 Hoe laat…?（幾點）→ om + 時間

講時間用 om + 點鐘。

🇳🇱 Hoe laat begint uw dag? → Ik sta op om acht uur. Ik drink dan eerst koffie.

你幾點開始一天？→ 我八點起床，會先喝咖啡。

M7 否定：niet vs geen

整句/動詞否定用 niet；名詞前否定用 geen。

🇳🇱 Ik reis niet met het vliegtuig. / Ik heb geen kinderen.

我不搭飛機。/ 我沒有小孩。

M8 Waar / Welke…? → 具體 + daar 延伸

給具體地名/名詞，再用 daar（在那裡）接一句延伸。

🇳🇱 Waar woont u? → Ik woon in Amsterdam. Daar woon ik al drie jaar.

你住哪？→ 我住阿姆斯特丹，我在那住三年了。
