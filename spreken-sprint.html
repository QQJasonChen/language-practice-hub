🏁 A2 口說衝刺包

167 題真考題 · 看題 → 🎙開口講(荷蘭文辨識) → 翻 model answer。標記不熟、只練弱項、隨機模擬考。

🧩 萬用答題模板（先記這個，任何題都能答）▾
核心公式：直接回答 ＋ 補一句（理由 / 頻率 / 地點 / 對象）
考官常追問「Vertel ook…」→ 就是要你多補一句。一律答 2 句最穩。
Wat…?
Ik [doe]… / Ik vind … leuk.
Wat doet u graag? → Ik kook graag. Ik vind dat leuk.
Hoe vaak…?
… [één/twee] keer per dag/week/maand/jaar.
Ik sport twee keer per week.
Waar…?
Ik [doe dat] in / bij / op …
Ik eet het liefst thuis. / in het park.
Welke / Wat voor…?
Ik houd van … Bijvoorbeeld …
Wat voor muziek? → Ik houd van popmuziek.
Waarom…?
…, want … （want 後面語序不變，最好用）
Ik fiets graag, want dat is gezond.
Met wie…?
Ik [doe dat] samen met mijn …
Ik eet samen met mijn familie.
Hoe…?
met de [fiets/auto] / eerst … dan …
Ik ga met de fiets. / Ik kook eerst de pasta, dan …
Wanneer / Hoe laat…?
Om [negen] uur. / In de ochtend · 's avonds.
Ik begin om negen uur.
加分小詞（塞進答案顯程度）：
het liefst（最愛）· graag（喜歡）· meestal（通常）· bijna altijd（幾乎總是）· soms（有時）· nooit（從不）· vaak（常）
表態：Ik vind … (niet) leuk / mooi / lekker / gezond / handig.
🖼️ 看圖題另有截圖拆解 → 看講義目錄。看圖題公式：先說人在做什麼（Hij/Zij [werkwoord]…）＋人在哪（Hij is in/op…）＋補你的看法。
