🗣 荷蘭文 A2 口說考古題講義

A2 口說考兩種題型：① 🔊 聽力→口說：聽考官問句，口頭回答（無圖）② 🖼️ 看圖回答問題：看螢幕上的圖回答（圖是關鍵）。兩種都用同一套答題模板。
📐 9 個萬用答題模板（先記）
T1Waarom…? → om … te / omdat（目的·原因）
T2帶介係詞問句 → er / ervan / er…naar
T3Ja/Nee 開頭 + 回扣問句動詞
T4頻率題 → iedere dag/week·keer per jaar
T5否定：niet（句/動詞）/ geen（名詞）
T6不會就：Nee, maar ik kan wel…
T7偏好：liever / het liefst / het leukst（+比較級 dan）
T8意見：Ik vind … + want/omdat
T9核心：答 2–3 句（直接答＋延伸）
🔊 題型一　聽力 → 口說
聽考官說一句＋問一個問題 → 你口頭回答 2-3 句。沒有圖，全靠聽。
第 1 題　🔊 聽力 → 口說
🎧 這題用聽的（無圖）— 先聽問句，再口頭回答

Hoe vaak gebruikt u een computer? Vertel ook waarvoor u een computer gebruikt.

你多常用電腦？也說用來做什麼。

✅ 範例答案

Ik gebruik elke dag een computer. Ik gebruik hem om te werken en om e-mails te sturen.

我每天用電腦，用來工作和寄郵件。

📌 這題在考什麼
兩段指令：Hoe vaak（頻率）+ Vertel ook waarvoor（用途） → 要答兩件事：頻率 + 用 om…te 講用途。
🧩 答題公式
Ik gebruik + 頻率 + …。Ik gebruik hem om + 用途 + te + 動詞。
🔍 拆解
  • elke dag 回答 hoe vaak
  • 聽到 Vertel ook 就一定要補第二句
  • om te werken / om e-mails te sturen 講用途
🔁 換情境也能套（依樣畫葫蘆）
om te studeren 為了讀書om films te kijken 為了看影片om met familie te bellen 為了跟家人通話
📖 重點詞
gebruiken 使用elke dag 每天om…te 為了…e-mails sturen 寄郵件
第 2 題　🔊 聽力 → 口說
🎧 這題用聽的（無圖）— 先聽問句，再口頭回答

Woont u liever in een drukke of in een rustige buurt? Vertel ook waarom.

你比較想住熱鬧還是安靜的區？也說原因。

✅ 範例答案

Ik woon liever in een rustige buurt, want dan kan ik goed slapen.

我比較想住安靜的區，因為這樣我能睡好。

📌 這題在考什麼
二選一(of) + liever（比較喜歡） + waarom → 選一個 + want 給理由。
🧩 答題公式
Ik … liever + 選項, want + 理由。
🔍 拆解
  • 先選一個（rustige buurt）
  • liever 表偏好
  • want 接理由（want 後語序不變）
🔁 換情境也能套（依樣畫葫蘆）
liever in een drukke buurt, want er is veel te doen 熱鬧、很多事做liever op het platteland 鄉下want het is er veiliger 比較安全
📖 重點詞
liever 比較喜歡drukke 熱鬧的rustige 安靜的buurt 社區want 因為
第 3 題　🔊 聽力 → 口說
🎧 這題用聽的（無圖）— 先聽問句，再口頭回答

Houdt u van fietsen? Vertel ook waarom u wel of niet van fietsen houdt.

你喜歡騎車嗎？也說喜歡或不喜歡的原因。

✅ 範例答案

Ja, ik hou van fietsen, want het is gezond en goed voor het milieu.

喜歡，因為健康又環保。

📌 這題在考什麼
Houdt u van…? + waarom → Ja/Nee + ik hou (er niet) van + want 理由。
🧩 答題公式
Ja, ik hou van + 名詞, want + 理由。
🔍 拆解
  • hou van + 名詞(fietsen)
  • want 給兩個理由更好(gezond en …)
  • 不喜歡：Nee, ik hou er niet van, want…
🔁 換情境也能套（依樣畫葫蘆）
want het is goedkoop 便宜Nee, want het regent vaak 常下雨want ik word er moe van 會累
📖 重點詞
houden van 喜歡fietsen 騎車gezond 健康het milieu 環境
第 4 題　🔊 聽力 → 口說
🎧 這題用聽的（無圖）— 先聽問句，再口頭回答

Wat eet u graag voor de lunch? Vertel ook met wie u meestal lunch eet.

你午餐喜歡吃什麼？通常和誰吃？

✅ 範例答案

Ik eet graag een broodje voor de lunch. Ik eet meestal samen met mijn collega's.

我午餐喜歡吃三明治，通常和同事一起吃。

📌 這題在考什麼
graag（喜歡做某事） + met wie（和誰） → 答喜歡吃啥 + 跟誰(samen met)。
🧩 答題公式
Ik eet graag + 食物。Ik eet samen met + 人。
🔍 拆解
  • graag 放動詞後
  • 聽到 Vertel ook met wie 要補人
  • samen met mijn collega's 和同事
🔁 換情境也能套（依樣畫葫蘆）
rijst met groente 菜飯samen met mijn familie 和家人alleen 一個人
📖 重點詞
graag 喜歡broodje 三明治meestal 通常samen met 和…一起
第 5 題　🔊 聽力 → 口說
🎧 這題用聽的（無圖）— 先聽問句，再口頭回答

Welk seizoen vindt u het leukst? Wat doet u graag in dat seizoen?

你最喜歡哪個季節？那季節喜歡做什麼？

✅ 範例答案

Ik vind de zomer het leukst. In de zomer ga ik graag naar het strand.

我最喜歡夏天。夏天我喜歡去海邊。

📌 這題在考什麼
het leukst（最…，最高級） + 延伸活動 → 選一個 + 那季節做什麼(graag)。
🧩 答題公式
Ik vind + X + het leukst. In + X + 動詞 + graag…
🔍 拆解
  • het leukst 最高級
  • 用 In de zomer 回扣季節再延伸
  • ga graag naar het strand 講活動
🔁 換情境也能套（依樣畫葫蘆）
de winter het leukst, want ik hou van sneeuw 冬天愛雪de lente 春天ga ik graag wandelen 喜歡散步
📖 重點詞
seizoen 季節het leukst 最喜歡de zomer 夏天het strand 海灘
第 6 題　🔊 聽力 → 口說
🎧 這題用聽的（無圖）— 先聽問句，再口頭回答

Hoe vaak gaat u naar de tandarts? En hoe voelt u zich dan?

你多常看牙醫？那時你感覺如何？

✅ 範例答案

Ik ga twee keer per jaar naar de tandarts. Ik voel me dan een beetje zenuwachtig.

我一年看兩次牙醫，那時有點緊張。

📌 這題在考什麼
頻率 + 感受(hoe voelt u zich) → 用 keer per jaar + ik voel me + 形容詞。
🧩 答題公式
Ik ga + X keer per jaar…。Ik voel me + 形容詞。
🔍 拆解
  • twee keer per jaar 一年兩次
  • voelen 是反身：ik voel me…
  • een beetje 稍微，緩和語氣
🔁 換情境也能套（依樣畫葫蘆）
één keer per jaar 一年一次ik voel me rustig 平靜een beetje bang 有點怕
📖 重點詞
tandarts 牙醫keer per jaar 每年…次zich voelen 感覺zenuwachtig 緊張
第 7 題　🔊 聽力 → 口說
🎧 這題用聽的（無圖）— 先聽問句，再口頭回答

Welke baan lijkt u leuker, in een ziekenhuis of in een winkel?

哪份工作你覺得比較有趣，醫院還是商店？（二選一）

✅ 範例答案

De baan in het ziekenhuis lijkt me leuker. Ik help graag andere mensen.

醫院那份比較有趣，我喜歡幫助別人。

📌 這題在考什麼
二選一比較題：leuker（比較…，比較級） → 選一個 + 一句理由。
🧩 答題公式
De … lijkt me leuker. + 理由。
🔍 拆解
  • lijkt me leuker = 我覺得比較有趣
  • comparatief：leuk→leuker
  • 加一句理由(Ik help graag…)
🔁 換情境也能套（依樣畫葫蘆）
lijkt me leuker, want ik hou van contact 喜歡跟人接觸in een winkel, want het is gezellig 商店、氣氛好
📖 重點詞
baan 工作lijken 看起來/覺得leuker 比較有趣ziekenhuis 醫院
第 8 題　🔊 聽力 → 口說
🎧 這題用聽的（無圖）— 先聽問句，再口頭回答

Waar woont u liever, in de stad of op het platteland?

你比較想住城市還是鄉下？（二選一）

✅ 範例答案

Ik woon liever in de stad, want daar is alles dichtbij.

我比較想住城市，因為那裡什麼都很近。

📌 這題在考什麼
二選一 + liever → 選一個 + want + daar 延伸。
🧩 答題公式
Ik woon liever + 地點, want daar + …
🔍 拆解
  • liever 偏好
  • want daar … 用 daar 回指地點再延伸
  • alles dichtbij 什麼都近
🔁 換情境也能套（依樣畫葫蘆）
op het platteland, want het is rustig 鄉下、安靜want daar is meer natuur 比較多自然
📖 重點詞
liever 比較想de stad 城市het platteland 鄉下dichtbij 近
第 9 題　🔊 聽力 → 口說
🎧 這題用聽的（無圖）— 先聽問句，再口頭回答

Wat vindt u moeilijk om te leren? Hoe lang probeert u dit al te leren?

你覺得學什麼很難？學多久了？

✅ 範例答案

Ik vind de Nederlandse grammatica moeilijk om te leren. Ik probeer dit al twee jaar te leren.

我覺得荷蘭語文法很難學。我已經學兩年了。

📌 這題在考什麼
vind … moeilijk + om te leren + 時間長度 → 意見 + 學多久(al + 時間)。
🧩 答題公式
Ik vind + X + moeilijk om te leren. Ik probeer dit al + 時間 + te leren.
🔍 拆解
  • vind X + 形容詞
  • om te leren 不定式
  • al twee jaar 已經兩年(al=已經)
🔁 換情境也能套（依樣畫葫蘆）
de uitspraak moeilijk 發音難de woorden 單字al zes maanden 已六個月
📖 重點詞
moeilijk 難grammatica 文法proberen 嘗試al 已經
第 10 題　🔊 聽力 → 口說
🎧 這題用聽的（無圖）— 先聽問句，再口頭回答

Maakt u wel eens muziek?

你偶爾會玩音樂嗎？

✅ 範例答案

Nee, ik maak zelf geen muziek. Maar ik luister er wel graag naar.

不，我自己不玩音樂。但我很喜歡聽。

📌 這題在考什麼
做不到 → Nee … geen …, maar … wel 轉折補一個會/喜歡的。
🧩 答題公式
Nee, ik … geen …. Maar ik … er wel … naar.
🔍 拆解
  • geen muziek 否定名詞
  • maar … wel 轉折
  • er … naar = luisteren naar 的代副詞
🔁 換情境也能套（依樣畫葫蘆）
maar ik zing wel graag 但我愛唱Ja, ik speel gitaar 我彈吉他maar ik dans wel 但我會跳舞
📖 重點詞
muziek maken 玩音樂geen 沒有wel 確實er…naar 聽它
第 11 題　🔊 聽力 → 口說
🎧 這題用聽的（無圖）— 先聽問句，再口頭回答

Wat staat er in uw koelkast? En wat bewaart u niet in de koelkast?

你冰箱裡有什麼？什麼不放冰箱？

✅ 範例答案

In mijn koelkast staan melk, kaas en groenten. Brood bewaar ik niet in de koelkast.

冰箱裡有牛奶、起司、蔬菜。麵包我不放冰箱。

📌 這題在考什麼
列舉 + 否定(niet) → 先列東西(staan)，再講什麼 niet。
🧩 答題公式
In mijn koelkast staan + 列舉。+ X bewaar ik niet…
🔍 拆解
  • staan 用於擺著的東西
  • 列三樣最佳(A, B en C)
  • bewaren niet 否定動作用 niet
🔁 換情境也能套（依樣畫葫蘆）
fruit en eieren 水果和蛋yoghurt 優格rijst bewaar ik niet 米不放
📖 重點詞
koelkast 冰箱staan 放著bewaren 保存groenten 蔬菜
第 12 題　🔊 聽力 → 口說
🎧 這題用聽的（無圖）— 先聽問句，再口頭回答

Hoe komt u naar Nederland? En met wie?

你怎麼來荷蘭的？跟誰？

✅ 範例答案

Ik kom met het vliegtuig naar Nederland. Ik kom samen met mijn man.

我搭飛機來荷蘭，和我先生一起來。

📌 這題在考什麼
交通方式(met) + met wie → met het + 交通工具 + samen met + 人。
🧩 答題公式
Ik kom met + 交通 + naar…。Ik kom samen met + 人。
🔍 拆解
  • met het vliegtuig 搭飛機(交通用 met)
  • samen met mijn man 和先生一起
  • 聽到 En met wie 要補人
🔁 換情境也能套（依樣畫葫蘆）
met de trein 火車met de boot 船samen met mijn kinderen 和小孩
📖 重點詞
het vliegtuig 飛機met 搭乘/和samen met 和…一起
🖼️ 題型二　看圖回答問題
螢幕出現一張圖 → 聽問題 → 根據圖回答。下面每題都附真考截圖。
第 1 題　🖼️ 看圖回答問題

Waarom sport je?

你為什麼運動？

✅ 範例答案

Ik sport om gezond te blijven.

我運動是為了保持健康。

📌 這題在考什麼
Waarom（為什麼）開頭 → 一定要給原因／目的。最常用「om … te …」表「為了…」。
🧩 答題公式
Ik + 動詞 + om + 目的 + te + 原形動詞
🔍 拆解
  • Ik sport — 回扣問句動詞 sport
  • om … te blijven —「為了…」框架，te+原形(blijven)放句尾
  • gezond（健康的）夾中間當目的
🔁 換情境也能套（依樣畫葫蘆）
om af te vallen 為了減重om fit te blijven 保持精實om te ontspannen 為了放鬆
📖 重點詞
sporten 運動gezond 健康的blijven 保持om…te 為了…
第 2 題　🖼️ 看圖回答問題

Waarom ga je naar de markt?

你為什麼去市場？

✅ 範例答案

Ik ga naar de markt om vis te kopen.

我去市場是為了買魚。

📌 這題在考什麼
一樣是 Waarom → 用 om…te 講目的；地點 naar de markt 要回扣。
🧩 答題公式
Ik ga naar + 地點 + om + 東西 + te + 動詞
🔍 拆解
  • Ik ga naar de markt — 回扣 ga + naar de markt
  • om vis te kopen — 為了買魚，te kopen 放句尾
🔁 換情境也能套（依樣畫葫蘆）
om groente te kopen 為了買菜om brood te halen 為了買麵包om mijn vriend te zien 為了見朋友
📖 重點詞
de markt 市場vis 魚kopen 買halen 取/買
第 3 題　🖼️ 看圖回答問題

Luister je vaak naar muziek?

你常聽音樂嗎？

✅ 範例答案

Ja, ik luister er iedere dag naar.

是的，我每天都聽。

📌 這題在考什麼
問句帶介係詞 naar 又問 vaak（常不常） → 答案要用「er … naar」代副詞 + 頻率詞。
🧩 答題公式
Ja, ik + 動詞 + er + 頻率 + naar
🔍 拆解
  • er 取代 muziek（被問的東西）
  • naar 放句尾（luisteren naar 的 naar）
  • iedere dag — 頻率詞回答 vaak
🔁 換情境也能套（依樣畫葫蘆）
er soms naar 偶爾聽er nooit naar 從不聽er elke avond naar 每晚聽
📖 重點詞
luisteren naar 聽muziek 音樂iedere dag 每天er…naar 代「聽它」
第 4 題　🖼️ 看圖回答問題

Hou je van pizza?

你喜歡披薩嗎？

✅ 範例答案

Ja, ik hou ervan.

是的，我喜歡。

📌 這題在考什麼
Hou je van …? 是高頻句型 → 答案用 ervan（= van + 它）。喜歡=hou ervan，不喜歡=hou er niet van。
🧩 答題公式
Ja, ik hou ervan. / Nee, ik hou er niet van.
🔍 拆解
  • houden van 喜歡
  • ervan = er + van，取代 pizza
  • 否定時 niet 插中間：er niet van
🔁 換情境也能套（依樣畫葫蘆）
Ja, ik hou ervan, vooral van Italiaanse pizza 尤其義式Nee, ik hou er niet van 不喜歡
📖 重點詞
houden van 喜歡ervan 喜歡它vooral 尤其
第 5 題　🖼️ 看圖回答問題

Koop je wel eens iets op de markt?

你偶爾會在市場買東西嗎？

✅ 範例答案

Ja, ik koop er vis en noten.

會，我在那買魚和堅果。

📌 這題在考什麼
wel eens（偶爾）的 yes/no 題 → Ja 開頭，用 er（在那裡）+ 具體買的東西。
🧩 答題公式
Ja, ik + 動詞 + er + 具體東西
🔍 拆解
  • er 取代 op de markt（地點）
  • vis en noten — 給具體例子比較分
🔁 換情境也能套（依樣畫葫蘆）
er groente en fruit 蔬果er kaas 起司er bloemen 花
📖 重點詞
wel eens 偶爾noten 堅果er 在那裡

第 7 題　🖼️ 看圖回答問題

Kun je goed dansen?

你舞跳得好嗎？

✅ 範例答案

Nee, maar ik kan wel goed zingen.

不，但我歌唱得不錯。

📌 這題在考什麼
做不到的事別只說 Nee → 用「Nee, maar ik kan wel …」轉折補一個會的，展現語言能力。
🧩 答題公式
Nee, maar ik kan wel + goed + 另一動詞
🔍 拆解
  • Nee — 先表態
  • maar … wel — 轉折，wel 強調「確實會」
  • kan + 原形(zingen)
🔁 換情境也能套（依樣畫葫蘆）
maar ik kan wel goed koken 會煮飯maar ik kan wel goed tekenen 會畫畫maar ik leer het graag 但我很想學
📖 重點詞
dansen 跳舞zingen 唱歌wel 確實maar 但是
第 8 題　🖼️ 看圖回答問題

Kom je uit Nederland?

你來自荷蘭嗎？

✅ 範例答案

Nee, ik kom niet uit Nederland.

不，我不是來自荷蘭。

📌 這題在考什麼
否定一個動作/整句 → 用 niet（放在要否定的部分前）。
🧩 答題公式
Nee, ik + 動詞 + niet + 其餘
🔍 拆解
  • komen uit 來自
  • niet 否定整個 uit Nederland
  • 可再加：Ik kom uit Taiwan 補出處
🔁 換情境也能套（依樣畫葫蘆）
Nee, ik kom uit Taiwan 我來自台灣Ja, ik kom uit Nederland 是的
📖 重點詞
komen uit 來自niet 不uit 從

Heeft u kinderen?

✅ 範例答案

Nee, ik heb geen kinderen.

不，我沒有小孩。

📌 這題在考什麼
否定一個名詞 → 用 geen（不是 niet！）。geen kinderen = 沒有小孩。
🧩 答題公式
Nee, ik heb geen + 名詞
🔍 拆解
  • 否定名詞 kinderen 用 geen
  • 對比：否定動詞才用 niet
🔁 換情境也能套（依樣畫葫蘆）
Ja, ik heb twee kinderen 有兩個Nee, ik heb geen broers 沒有兄弟
📖 重點詞
kinderen 小孩geen 沒有(名詞)heb 有

第 11 題　🖼️ 看圖回答問題

Waarom wil je in de bibliotheek studeren?

你為什麼想在圖書館念書？

✅ 範例答案

Omdat het rustig is.

因為很安靜。

📌 這題在考什麼
Waarom 也可用 Omdat（因為）回答 → omdat 是從屬連接詞，動詞要丟到句尾。
🧩 答題公式
Omdat + 主詞 + … + 動詞(句尾)
🔍 拆解
  • Omdat het rustig is — is 放句尾(omdat 後動詞後置)
  • 比 om…te 更口語自然講原因
🔁 換情境也能套（依樣畫葫蘆）
Omdat ik me kan concentreren 因為能專心Omdat er veel boeken zijn 因為書多Omdat het er stil is 因為很安靜
📖 重點詞
omdat 因為rustig 安靜studeren 念書bibliotheek 圖書館
第 12 題　🖼️ 看圖回答問題

Hou je van Nederlandse kaas?

你喜歡荷蘭起司嗎？

✅ 範例答案

Nee, ik hou er niet van.

不，我不喜歡。

📌 這題在考什麼
又是 Hou je van…? → 否定版：Ik hou er niet van（er + niet + van）。
🧩 答題公式
Nee, ik hou er niet van.
🔍 拆解
  • ervan 拆開成 er … van
  • niet 插中間否定
  • 可加理由：want het smaakt te sterk 味道太重
🔁 換情境也能套（依樣畫葫蘆）
Ja, ik hou ervan 喜歡Nee, ik hou er niet zo van 不太喜歡
📖 重點詞
kaas 起司Nederlandse 荷蘭的er niet van 不喜歡它
💡 萬用流程：聽問句抓關鍵字（Waarom/vaak/Hou je van/liever…）→ 對到模板 → 答「直接答＋一句延伸」。看圖題先判斷圖的主題再套。圖會變、公式不變。
